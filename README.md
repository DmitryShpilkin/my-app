# Проект: Страница регистрации с валидацией

Проект, реализующий страницу регистрации на React с использованием TypeScript. Включает в себя типизированные компоненты, простую валидацию формы и набор тестов.

## 🛠 Технологии

*   **React** — Библиотека для создания пользовательских интерфейсов.
*   **TypeScript** — Язык со строгой типизацией для надежности кода.
*   **Jest + React Testing Library** — Инструменты для написания и запуска тестов.
*   **simple-react-validator** — Библиотека для простой валидации форм.
*   **react-router-dom** — Библиотека для навигации и маршрутизации.

## 🚀 Запуск проекта

1.  **Клонируйте репозиторий** и перейдите в папку проекта:
    ```bash
    git clone <[https://github.com/DmitryShpilkin/my-app](https://github.com/DmitryShpilkin/my-app)>
    cd <my-app>
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    # или
    yarn install
    ```

3.  **Запустите приложение:**
    ```bash
    npm start
    # или
    yarn start
    ```

4.  **Откройте в браузере:**
    Приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000).

## 🧪 Запуск тестов

1.  **Выполните команду для запуска тестов:**
    ```bash
    npm test
    # или
    yarn test
    ```
2.  **Проверьте результат:**
    Убедитесь, что в консоли все тесты отмечены как **`PASS`** (зеленый цвет).

## 📂 Структура проекта

*   `src/components/RegistrationForm.tsx` — Компонент формы регистрации с логикой и валидацией.
*   `src/api/auth.ts` — API-заглушка для имитации отправки данных на сервер.
*   `src/components/index.ts` — Реэкспорт компонентов для удобства импорта.
*   `src/components/RegistrationForm.test.tsx` — Юнит-тесты для формы регистрации.
*   `src/components/Header.tsx` — Навигационный хедер со ссылками на страницы.
*   `src/components/LoginForm.tsx` — Компонент формы логина (заглушка)
